# Markdown Editor
[demo](https://boseman1024.github.io/MarkdownEditor/)
* 实时预览
* ~~同步滚动~~
* 代码高亮
* 工具栏基本功能
* 快捷键操作
    `Ctrl+B`加粗
    `Ctrl+G`图片
    `Ctrl+l`链接
    `Ctrl+K`代码块
    `Ctrl+Q`引用
    `Ctrl+Z`撤销
